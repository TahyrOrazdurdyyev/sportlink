# Настройка MongoDB для Sportlink

## Установка MongoDB

### Windows:
1. Скачайте MongoDB Community Server: https://www.mongodb.com/try/download/community
2. Запустите установщик
3. Выберите "Complete" установку
4. Установите MongoDB Compass (GUI инструмент)

### Или через Docker:
```bash
docker run --name sportlink-mongo -p 27017:27017 -d mongo:latest
```

## Создание базы данных

MongoDB создаст базу автоматически при первом использовании.

### Данные подключения:
```
База данных: sportlink
Хост: localhost
Порт: 27017
Пользователь: (не требуется для локальной разработки)
Пароль: (не требуется для локальной разработки)
```

### Connection String:
```
mongodb://localhost:27017/sportlink
```

## Преимущества для проекта:

✅ **Геолокация**: Встроенная поддержка 2dsphere индексов
✅ **JSON/JSONB**: Нативная работа с документами  
✅ **i18n**: Идеально для многоязычных полей
✅ **Гибкость**: Легко менять структуру данных
✅ **Производительность**: Быстрые геозапросы
✅ **Масштабирование**: Горизонтальное масштабирование

## Что изменится:
- Модели переписаны под mongoengine
- Геолокационные запросы через MongoDB GeoJSON
- Нет Django migrations (MongoDB schema-less)
- Более гибкая работа с вложенными данными
